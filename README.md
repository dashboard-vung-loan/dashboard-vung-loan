# YODY Dashboard Realtime - Vùng Loan

## Cách dùng nhanh
1. Mở `index.html` để xem dashboard mẫu.
2. Cập nhật dữ liệu trong các file CSV:
   - `tong_quan.csv`
   - `cua_hang.csv`
   - `hang_hoa.csv`
   - `action.csv`

## Cách làm realtime bằng Google Sheet
1. Upload file Excel mẫu lên Google Drive.
2. Mở bằng Google Sheets.
3. Tạo từng tab dữ liệu giống tên cột trong file CSV.
4. Vào File → Share → Publish to web → chọn từng sheet → CSV.
5. Copy link CSV và thay vào đầu file `script.js`.

## Đưa lên GitHub Pages
1. Tạo repository mới trên GitHub.
2. Upload toàn bộ file trong thư mục này.
3. Vào Settings → Pages → Deploy from branch → main → root.
4. GitHub sẽ tạo link dashboard online.
